# Use 
to use the Project and the build tools you need to download the gowin-eda from there website and place the IDE Directory in the root Directory of this Project then just run `make env` to build an run the Docker-Container. To do this you need to write the command `export QT_QPA_PLATFORM=offscreen` and then run `python3 cpu.py --build` after this the Project will get build.
